Financial Formulas

Net Present Value:

\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \]

Where \(CF_t\) = cash flows, \(r\) = discount rate, \(C_0\) = initial investment.

Cost of Equity (CAPM):

\[ k_e = r_f + \beta (r_m - r_f) \]

Explain sensitivity to discount rate and beta.

Python Check: Consensus vs Actuals


from dataclasses import dataclass

@dataclass
class KPIs:
    revenue: float
    margin: float
    eps: float

k = KPIs(1240.0, 0.184, 1.12)

def beats_consensus(actual, consensus):
    return all(a >= c for a, c in zip(actual, consensus))

print(beats_consensus(
    (k.revenue, k.margin, k.eps),
    (1215.0, 0.175, 1.08)
))  # True
