Financial Math

Net Present Value (NPV):

\[ \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \]

Where \(CF_t\) are cash flows, \(r\) is discount rate, and \(C_0\) is initial cost.

Cost of Equity (CAPM):

\[ k_e = r_f + \beta (r_m - r_f) \]

Mention sensitivity analysis on r and T. Show how guidance ranges shift NPV and k_e.

Tech Highlight: Programmatic KPI Check

from dataclasses import dataclass

@dataclass
class KPIs:
    revenue: float  # in millions
    op_margin: float  # 0..1
    eps: float

k = KPIs(revenue=1240.0, op_margin=0.184, eps=1.12)

def beats_consensus(actual, consensus):
    return all(a >= c for a, c in zip(actual, consensus))

print(beats_consensus(
    actual=(k.revenue, k.op_margin, k.eps),
    consensus=(1215.0, 0.175, 1.08)
))  # True
